Kevert Cheatsheet

HTML

Bekezdés:

<p>Ez egy bekezdés.</p>

Link:

<a href="https://www.example.com">Ez egy link</a>

Kép:

<img src="image.jpg" alt="Leírás">

CSS

Szöveg színezése:

.text-red {
    color: red;
}

Háttér színezése:

.bg-blue {
    background-color: blue;
}

Szöveg igazítása:

.text-center {
    text-align: center;
}

JavaScript

Változók:

let name = "John";
const pi = 3.14;

Függvény:

function greet() {
    console.log("Hello, World!");
}

Event Listener:

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

jQuery

Document Ready:

$(document).ready(function() {
    console.log("Document ready!");
});

Element kiválasztása és módosítása:

$("#myElement").text("Hello, World!").css("color", "red");

Eseménykezelés:

$("#myButton").click(function() {
    alert("Button clicked!");
});
