Mértékegységek

  • rem: A gyökérelem (root) betűmérete, 1 rem = 16px alapértelmezés szerint.
  • px: Pixel, egy képpont a kijelzőn.
  • em: Az aktuális elem betűmérete, relatív az elem szülőjének betűméretéhez.
  • %: Százalékos arány, az elem szülőjének méretéhez viszonyítva.
  • vw: Viewport szélességének százaléka.
  • vh: Viewport magasságának százaléka.
  • fr: A rács (grid) elosztási egysége, pl. 1fr = 1/3 szélesség három oszlop esetén.

Grid Rendszer

.container: Fix szélességű tároló, középre igazítva.

.row: Rács sor, amely tartalmazza az oszlopokat.

.col: Alapértelmezett oszlop, amely egyenlő szélességű az összes többi oszloppal.

.col-{breakpoint}-{size}: Specifikus oszlop méret a különböző képernyő méretekhez (pl. col-md-4).
